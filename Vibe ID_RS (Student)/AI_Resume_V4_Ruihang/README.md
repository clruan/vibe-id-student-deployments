# AI Resume Webpage

Static interactive resume webpage for Ruihang Shen.

## Start

```powershell
npm start
```

Then open:

```text
http://127.0.0.1:5173/
```

If `5173` is already in use, the server prints the next available local URL.

No dependency install is required. The page uses local files plus CDN-hosted Three.js and GSAP scripts.

## Project Files

- `index.html` - page structure and script loading
- `assets/styles.css` - visual styling and responsive layout
- `assets/content.js` - resume content data
- `assets/app.js` and `assets/js/` - interactions, navigation, project demos, skill panels, and background animation
