# Vibe ID_RS

This folder is ready for local page editing.

## Run the Page

```bash
npm run dev
```

Then open the URL printed in the terminal, usually:

```text
http://127.0.0.1:5173/
```

## Edit These Files

- `AI_Resume_V4_Ruihang/index.html` - page structure
- `AI_Resume_V4_Ruihang/assets/styles.css` - visual styling
- `AI_Resume_V4_Ruihang/assets/content.js` - resume/profile content
- `AI_Resume_V4_Ruihang/assets/app.js` and `AI_Resume_V4_Ruihang/assets/js/` - interactions and page behavior

## Assets

User images, PDFs, and project source files live in `User_data/`.

The local server is configured so the page can reference those files from the neighboring `User_data` folder.
